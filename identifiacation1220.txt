========================================
予備同定実験 手順まとめ
（2入力遊星歯車型 可変減速アクチュエータ）
========================================

【位置づけ】
本実験は、可変減速自由度を用いた協調制御研究のための
「予備同定実験」であり、高精度モデルの構築を目的としない。
理想モデルと実機との差異のオーダを把握し、
提案制御の成立条件を確認することを目的とする。

----------------------------------------
1. 理論モデル（基準）
----------------------------------------

[運動学]
ω_out = a1 * ω1 + a2 * ω2

a1 = 1/20
a2 = 57/700

[主モータの実効減速比]
（主モータ：motor1、副モータ：motor2）

ρ = ω2 / ω1
i_eff(ρ) = ω1 / ω_out = 1 / (a1 + a2 * ρ)

本同定では、この関係が実機でどの程度成立するかを確認する。

----------------------------------------
2. 実験①：運動学係数の線形確認
----------------------------------------

【目的】
・運動学が線形結合として成立しているかを確認
・a1, a2 が理論値に近いかを確認
・摩擦・バックラッシュの影響のオーダ把握

【指令】
・motor1, motor2 ともに速度制御
・以下の組合せを順に与える（定常区間を使用）

  ω1 ∈ {±5, ±10, ±15} [rad/s]
  ω2 ∈ {±5, ±10, ±15} [rad/s]

・各条件で 1〜2 秒定常保持

【計測】
・ω1（主モータ角速度）
・ω2（副モータ角速度）
・ω_out（出力角速度）
・時刻 t

【同定計算】
ω_out ≈ a1 * ω1 + a2 * ω2

最小二乗法により a1, a2 を推定。
定常区間のみを使用（加速中データは除外）。

【得られる結果】
・a1, a2 の実測値
・理論値との差（%）
・低速域での誤差増大の有無

----------------------------------------
3. 実験②：主モータ実効減速比の可変確認
----------------------------------------

【目的】
・副モータ動作によって
  主モータから出力までの減速比が変化することを実機で確認
・減速比調整が有効に機能する速度域の確認

【指令】
・主モータ速度一定：
  ω1 = ω1_ref（例：+10 rad/s）

・副モータ速度を比率で変更：
  ρ ∈ {-1.0, -0.5, 0, +0.5, +1.0, +1.5}
  ω2 = ρ * ω1

・各条件で 1〜2 秒定常保持

【計測】
・ω1, ω2, ω_out

【計算】
i_eff = ω1 / ω_out

理論値：
i_eff_theory = 1 / (a1 + a2 * ρ)

【得られる結果】
・i_eff vs ρ の実測曲線
・理論曲線との比較
・減速比変化が摩擦に埋もれる領域／有効な領域

----------------------------------------
4. 実験③：摩擦特性の簡易把握（主モータ単体）
----------------------------------------

【目的】
・主モータ側摩擦（特にクーロン・粘性）のオーダ把握
・減速比調整効果が摩擦に勝つ条件の確認

【指令】
・副モータ停止（ω2 = 0）
・主モータを定速駆動：

  ω1 ∈ {±2, ±5, ±10, ±15} [rad/s]

・各条件で 1〜2 秒定常保持

【計測】
・ω1
・主モータ電流 Iq1
・トルク換算値 τ1 = k_t * Iq1

【同定モデル（簡易）】
τ_fric(ω) ≈ τ_c * sgn(ω) + b * ω

最小二乗法で τ_c, b を推定。

【得られる結果】
・クーロン摩擦・粘性摩擦のオーダ
・低速域での摩擦支配の有無

----------------------------------------
5. 実験④：バックラッシュ存在確認（参考）
----------------------------------------

【目的】
・遊星機構内バックラッシュの有無と影響の確認
・低速域で理論式が成立しない理由の把握

【指令（例）】
・副モータ固定（ω2 = 0）
・主モータ速度を極低速で往復：

  ω1 ≈ ±0.2〜0.5 rad/s

【計測】
・主モータ角度
・出力角度（可能であれば）

【評価】
・主角度が変化しても出力が動かない区間の有無
・定量化は行わず、存在確認のみ

----------------------------------------
6. 本研究における同定結果の扱い
----------------------------------------

【主張してよいこと】
・理論モデルとの差の傾向
・副モータが減速比調整自由度として機能する条件
・提案制御が有効となる速度・負荷領域

【主張しないこと】
・高精度モデル
・一般化可能な同定手法
・摩擦・慣性の厳密同定

----------------------------------------
7. まとめ（研究との関係）
----------------------------------------

本同定実験により、
副モータによって主モータの実効減速比が変化し得ること、
およびその効果が摩擦・速度条件に依存することを確認する。

これらの結果を前提として、
本研究では減速比調整自由度を用いた
主モータ優先・トルク余裕最大化制御の設計と評価を行う。
